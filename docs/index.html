QGIS 3.44 ドキュメント

ヒント

3.44 Long Term Release よりも新しいバージョンのドキュメントをお探しでしょうか？ その場合には、 testing docs を見てください。

注釈

QGISドキュメントは様々な言語やバージョンについて利用可能です。サイドバー下部にある QGIS Documentation メニューを展開すると、一覧が表示されます。

注釈

このドキュメントは Transifex のコミュニティメンバーによって元の英語版から翻訳されています。

翻訳作業の 完了レベル によっては、まだ英語のままの段落やページがあるかもしれません。Transifexで新たに翻訳文を作成したり、既存の翻訳をレビューすることで、コミュニティを支援することができます。

現時点では、ローカライズされた翻訳は Long Term リリースブランチでのみ利用可能ですが、QGIS の使い方を学ぶには十分対応できるはずです。

フリーでオープンソースのコミュニティ主導 GIS ソフトウェアである QGIS の公式ドキュメントへようこそ！このドキュメントを初めてご覧になる方は、以下および左側のサイドバーにある目次を使うことで、興味のあるトピックのドキュメントに簡単にアクセスできます。左上にある検索機能も利用できます。

オンラインドキュメントの他に、QGISプロジェクトではダウンロードしてオフラインで読むことができる資料も用意しています。資料はサイドバー下部にある QGIS Documentation メニューからアクセスでき、以下の形式があります：

  • zip圧縮されたHTMLファイル。展開して ドキュメントへのパス を通せば、ソフトウェア内からも利用可能です。

  • PDFファイル

注釈

QGIS は、コミュニティによって開発されているオープンソースプロジェクトです。ドキュメントチームは、チュートリアルや機能の説明を改善するために、皆さんからのフィードバックや協力を常に必要としています。理解しにくい部分があったり、ドキュメント内で探しているものが見つからない場合は、お知らせいただくことでドキュメントの改善にご協力ください:

  • GitHubリポジトリ にイシューやプルリクエストを提出する

  • あなたの言語に翻訳を行って translate the documentation 私たちを助けて下さい

  • 私たちと会話したい場合は qgis-community-team メーリングリスト, IRCの #qgis チャンネルまたは #qgis:matrix.org ルームでお気軽に声をかけて下さい!

以下のいずれかのドキュメントをご覧ください。
