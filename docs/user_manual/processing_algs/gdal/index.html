24.2. GDAL アルゴリズムプロバイダ

GDAL (Geospatial Data Abstraction Library) は、ラスタとベクタの地理空間データ形式の翻訳ライブラリです。プロセシングフレームワークのアルゴリズムは GDAL raster programs と GDAL vector programs から派生したものです。
