QGISは、Open Source Geospatial Foundation（OSGeo）傘下のオープンソースプロジェクトであり、熱心なボランティアや組織のチームによって開発されています。私たちは、人種、信条、性別、そしてあらゆる背景を持つ人々が歓迎されるコミュニティを目指しています。あなたはいつでも 参加する ことができます。

翻訳者
