17.2. 始める前の重要な警告

ワードプロセッサのマニュアルが小説や詩を書く方法を教えてくれないように、CADチュートリアルが建物の梁のサイズを計算する方法を示してくれないように、このガイドでは空間分析を教えません。代わりに、空間分析を行うための強力なツールであるQGISプロセシングフレームワークを使う方法を説明します。その種の分析を理解するために必要な概念を学ぶかどうかはあなた次第です。試してみたくなるかもしれませんが、それらなしでは、フレームワークとそのアルゴリズムを使っても何にもなりません。

例を挙げてもっと明確に説明しましょう。

与えられた点の集合と、各点における与えられた変数の値から クリギング ジオアルゴリズムを使ってラスタレイヤを計算できます。そのモジュールのパラメータダイアログは、次のようなものです。

それは複雑に見えます、よね？

このマニュアルを読むことで、そのモジュールの使い方、バッチ処理で数百のポイントレイヤから一度にラスタレイヤを作成する方法、入力レイヤに選択されているポイントがある場合はどうなるかなど、さまざまなことを学ぶことができます。ただし、パラメータそのものは説明されていません。地理統計学に精通した経験豊富な分析者であれば、これらのパラメータを理解するのに何の問題もないでしょう。もしあなたがそうではなく、 シル 、 レンジ 、または ナゲット という概念に馴染みがなければ、 クリギング モジュールを使うべきではありません。さらに言えば、クリギング モジュールを使うには、空間自己相関やセミバリオグラムといった、おそらく聞いたことがないか、十分に勉強していない概念について学ぶ必要があるので、使う準備ができているとは言えません。まずはそれらを学習し理解した上で、QGISに戻って実際に実行し、分析を行うべきです。この順序を無視すると、間違った結果や貧弱な（そしてほとんどの場合役に立たない）分析結果を招くことになります。

全てのアルゴリズムがクリギングのように複雑ではありません（しかし、さらに複雑なものもあります！）が、ほとんどはそれが基盤としている基本的な分析の考え方を理解することが必要です。その知識がなく使えば、ほぼ確実に悪い結果を招くでしょう。

空間分析の基礎知識を持たずにジオアルゴリズムを使うことは、文法や構文の知識がなく、物語の構成についても何も知らない状態で小説を書こうとするようなものです。結果は得られるかもしれませんが、その価値はほとんどないでしょう。どうか自分を欺かず、このガイドを読んだだけで空間分析を行って健全な結果が得られるようになると考えないでください。空間分析についても学ぶ必要があります。

ここに、空間データ分析についての詳細を学ぶために読むことができる良い参考文献があります。

Geospatial Analysis (3rd Edition): A Comprehensive Guide to Principles, Techniques and Software Tools Michael John De Smith, Michael F. Goodchild, Paul A. Longley

それは ここ でオンラインで入手できます
